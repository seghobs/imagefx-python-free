ImageFX API

Generate AI images programmatically

Generate Images

Generate AI images using a simple GET request.

Endpoint

GET /api/generate

Parameters

prompt (required) - Text description of the image to generate
count (optional) - Number of images to generate (1-8, default: 4)
seed (optional) - Seed value for consistent results

Example Request

http://your-server/api/generate?prompt=a beautiful sunset over mountains&count=4&seed=12345

Example Response

{
    "success": true,
    "images": [
        {
            "url": "data:image/png;base64,...",
            "seed": 12345,
            "prompt": "a beautiful sunset over mountains"
        }
    ]
}

Error Response

{
    "error": "Error message here"
}

Usage Examples

JavaScript

async function generateImage(prompt) {
    const response = await fetch(`http://your-server/api/generate?prompt=${encodeURIComponent(prompt)}`);
    const data = await response.json();
    
    if (data.success) {
        data.images.forEach(image => {
            const img = document.createElement('img');
            img.src = image.url;
            document.body.appendChild(img);
        });
    } else {
        console.error(data.error);
    }
}

Python

import requests

def generate_image(prompt):
    response = requests.get(
        'http://your-server/api/generate',
        params={
            'prompt': prompt,
            'count': 4
        }
    )
    data = response.json()
    
    if data['success']:
        for image in data['images']:
            print(f"Image URL: {image['url']}")
            print(f"Seed: {image['seed']}")
    else:
        print(f"Error: {data['error']}")
